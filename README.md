## üìö Steps to Run the Project

1. **Clone the repository**

   ```bash
   git clone https://github.com/shivamangina/Galadriel-OnChain-LLM
   ```

2. **Install the dependencies**

   ```bash
   npm install
   ```

3. **Start the app**

   ```bash
   npm run callJokeGenerater
   ```
